# Detachable Trash Compactor (탈부착형 쓰레기 압축기)

**기존 쓰레기통에 바로 설치하는 고효율·저비용 압축 솔루션**

---

## 📱 프로젝트 소개

**Detachable Trash Compactor**는 고가의 스마트 압축 쓰레기통을 새로 구매하는 대신, **기존에 사용하던 공공장소 쓰레기통에 바로 장착**하여 효율적으로 부피를 줄이는 로봇 자동화 프로젝트입니다.

기존 상용화 모델(약 300만 원)의 높은 비용 문제를 해결하기 위해, **약 100만 원 안팎의 예산**으로 제작 가능한 탈부착 및 규격 조절형 장치를 설계하였습니다.

---

## 🧭 설계 목적

- **경제성 확보**: 대당 도입 비용을 기존 벤치마킹 모델 대비 약 1/3 수준으로 절감
- **범용성 극대화**: 조절 가능한 프레임을 통해 70L에서 145L까지 다양한 규격의 쓰레기통에 안정적으로 설치 가능
- **환경 및 관리 효율**: 쓰레기 부피 압축을 통해 수거 빈도를 80% 이상 감소시키고 주변 미관을 개선

---

## 🛠 기술 스택

### Hardware & Mechanism
- **Controller**: Arduino UNO R3 (제어 알고리즘 구현)
- **Actuator**: WGM58-BL4280 BLDC 모터 (24V, 정격 토크 16kgf·cm)
- **Power**: 24V 100,000mAh 리튬이온 배터리 (연속 모터 구동 약 22.5시간 가능)
- **Driver**: BLC-150 모터 드라이버 (12-36V, 최대 500W 출력 지원)
- **Transmission**: SFU-2010 볼스크류 (피치 10mm, 직선 운동 및 고하중 압축 구현)

### Framework & Design
- **Body**: 알루미늄 프로파일(4040) 및 사각 파이프(50 x 50 x 5T) 조합
- **Plate**: 지름 200mm 원뿔형 압축판 (낙하 방지용 수직 가이드 시스템 포함)

---

## 🧩 시스템 설계

### 정량적 설계 목표
- **압축력**: 최대 100kgf (모터 토크 및 볼스크류 피치를 고려한 수치)
- **가동 범위**: 최대 30cm (공공장소 쓰레기통 깊이 대응)
- **동작 시간**: 1회 압축 및 복귀 프로세스 30초 이내 완료

### 알고리즘 플로우
1. **대기 상태**: 사용자 버튼 입력 상시 감지
2. **압축 동작**: 모터 정방향 회전 → 압축판 하강 → 최하부 도달 시 방향 전환
3. **복귀 동작**: 모터 역방향 회전 → 압축판 상승 → 최상부 도달 시 자동 정지

---

## 🔄 주요 기술 적용 예시

### 모터 제어 (Arduino)
- `TimerOne.h` 라이브러리를 사용한 고주파 PWM 제어로 모터 소음 감소 및 정밀 속도 제어
- 물리적 버튼의 노이즈 제거를 위한 소프트웨어 디바운싱($50ms$) 적용

```cpp
// 버튼 입력 감지 시 동작 로직 예시
if (digitalRead(BUTTON_PIN) == LOW) {
    delay(50); // 디바운싱
    if (digitalRead(BUTTON_PIN) == LOW) {
        digitalWrite(DIR_PIN, LOW); // 정방향 하강
        Timer1.setPwmDuty(PWM_PIN, DUTY);
        delay(14500); // 14.5초간 하강
        
        digitalWrite(DIR_PIN, HIGH); // 역방향 복귀
        Timer1.setPwmDuty(PWM_PIN, DUTY);
        delay(14300); // 14.3초간 상승
        
        Timer1.setPwmDuty(PWM_PIN, 0); // 정지
    }
}
```

### 기구적 가이드 설계
- 볼스크류 회전 시 압축판이 함께 회전하는 현상을 방지하기 위해 수직 가이드를 추가로 장착하여 안정적인 직선 운동 구현

## 🚨 문제 상황 및 해결 (Troubleshooting)

### 압축 시 본체 이탈 현상 발생
실제 쓰레기통 설치 시현 중, 압축판이 쓰레기 뭉치를 누를 때 발생하는 수직 반발력으로 인해 압축기 몸체가 쓰레기통 위로 밀려 올라가는 문제를 확인했습니다.

### 🔍 해결 접근
- 단순 결합만으로는 쓰레기의 강한 저항(목표 $100kgf$)을 견디기에 마찰력이 부족함을 확인.
- 쓰레기통 벽면과 닿는 지지부의 접지력 향상 필요.

### ✅ 수정 방안
- 고무 패드 추가: 쓰레기통 옆면과 접촉하는 프레임 끝단에 고무 패드를 부착하여 마찰 계수 증대.
- 결과: 접지력이 향상되어 압축 시에도 본체가 흔들림 없이 고정됨을 확인.

## ✨ 배운 점 / 느낀 점
- 엔지니어링의 실무적 적용: 이론적으로 산출한 토크와 압축력이 실제 환경(쓰레기의 탄성, 본체 밀림 등)에서 어떻게 변수로 작용하는지 체감하며 설계를 보완함.
- 프로토타입 제작 역량: 알루미늄 프로파일과 사각 파이프를 활용하여 규격 조절이 가능한 견고한 프레임을 직접 제작하며 기구 설계 능력을 배양함.
- 하드웨어 통합 제어: 모터 드라이버, 배터리, MCU 간의 회로 구성 및 제어 코딩을 통해 시스템 통합 역량을 강화함.
